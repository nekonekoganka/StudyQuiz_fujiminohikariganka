# カラーテーマ一覧

クイズカードの左ボーダーに使用する色の一覧です。
クイズの内容に合わせて選んでください。

---

## 使用可能なカラーテーマ

| テーマ名 | 色 | 使用例 | おすすめ用途 |
|---------|-----|--------|-------------|
| `blue` | 紺〜青 | 院内ルール、近視 | ルール系、基礎知識 |
| `cyan` | 水色〜青緑 | CL処方、白内障 | コンタクト系、手術系 |
| `teal` | ティール（青緑） | 花粉症 | アレルギー系 |
| `green` | 緑 | 緑内障 | 緑内障関連 |
| `amber` | オレンジ〜黄 | 老眼鏡、遠用メガネ | メガネ系 |
| `purple` | 紫 | 斜視、弱視 | 小児眼科系 |
| `indigo` | インディゴ（藍色） | 検診制度 | 制度・行政系 |

---

## カラーコード詳細

### blue（紺〜青）
```css
background: linear-gradient(180deg, #1e3c72, #2a5298);
```
- 落ち着いた印象
- 信頼感のある色
- 院内ルールなど堅めの内容に

### cyan（水色〜青緑）
```css
background: linear-gradient(180deg, #4facfe, #00f2fe);
```
- 爽やかな印象
- 清潔感のある色
- コンタクト、目の疾患に

### teal（ティール）
```css
background: linear-gradient(180deg, #4db6ac, #00796b);
```
- 自然な印象
- 癒しの色
- 花粉症などアレルギー系に

### green（緑）
```css
background: linear-gradient(180deg, #2e7d32, #1b5e20);
```
- 安心感のある色
- 緑内障に最適（名前の通り）

### amber（オレンジ〜黄）
```css
background: linear-gradient(180deg, #ffb300, #ff8f00);
```
- 温かみのある色
- 明るい印象
- メガネ系に

### purple（紫）
```css
background: linear-gradient(180deg, #ab47bc, #8e24aa);
```
- 優しい印象
- 小児眼科系に

### indigo（インディゴ）
```css
background: linear-gradient(180deg, #667eea, #764ba2);
```
- 知的な印象
- 制度や検診など

## 新しいクイズの色選びガイド

### カテゴリ別おすすめ

**メガネ・コンタクト系**
- コンタクト → `cyan`
- メガネ → `amber`

**眼の疾患系**
- 緑内障 → `green`
- 白内障 → `cyan`
- 近視・遠視 → `blue`
- アレルギー・花粉症 → `teal`
- 小児（弱視・斜視） → `purple`
- ドライアイ → `cyan` または `teal`
- 網膜疾患 → `indigo`
- 感染症（結膜炎等） → `teal`

**院内ルール・制度系**
- 院内ルール → `blue`
- 検診・制度 → `indigo`

## HTMLでの使い方

### index.html（トップページカード）

```html
<a href="quizzes/○○クイズ.html" class="quiz-card cyan" data-quiz-id="xxx">
```

`class="quiz-card cyan"` の `cyan` 部分を変更

### quiz-config.js

```javascript
{
    id: 'xxx',
    name: '○○クイズ',
    // ...
    color: 'cyan'  // ← ここを変更
},
```

## カスタムカラーを追加する場合

`index.html` の `<style>` セクションに以下を追加：

```css
.quiz-card.newcolor::before {
    background: linear-gradient(180deg, #色1, #色2);
}
```

ダークモード対応も忘れずに：

```css
@media (prefers-color-scheme: dark) {
    .quiz-card.newcolor::before {
        background: linear-gradient(180deg, #ダーク色1, #ダーク色2);
    }
}
```

## 色の選び方のコツ

1. **内容との関連性**
   - 緑内障 → 緑（green）
   - 花粉症 → 自然の色（teal）

2. **カテゴリの統一感**
   - 同じカテゴリは近い色系統に
   - メガネ系は暖色（amber）
   - 小児系は優しい色（purple）

3. **視認性**
   - 似た色が並ばないように
   - 区別しやすい色を選ぶ

4. **既存クイズとの重複**
   - 同じ色が多くなりすぎないように
   - バランスを見て選ぶ
