# 別AIへのプロンプト例

眼科クイズを作成する際に、別のAI（ChatGPT、Gemini、Perplexity等）に渡すプロンプトの例です。
コピペしてそのまま使えます。

---

## 基本プロンプト（URLから作成）

```
以下の仕様に従って、眼科スタッフ向けクイズを作成してください。

■ 参考ページ
URL: [ここにURLを貼り付け]

■ クイズ情報
- タイトル: [例: 流涙症クイズ]
- 問題数: 10問

■ 出力形式
以下のJSON形式で出力してください：

{
  "quizTitle": "クイズタイトル",
  "quizSubtitle": "眼科スタッフ向け - ○○の理解を深めよう",
  "quizIcon": "絵文字1つ",
  "quizId": "英語小文字のID（例: ruiruisho）",
  "totalQuestions": 10,
  "questions": [
    {
      "id": 1,
      "questionText": "問題文",
      "options": [
        {"text": "正解の選択肢", "isCorrect": true},
        {"text": "不正解の選択肢A", "isCorrect": false},
        {"text": "不正解の選択肢B", "isCorrect": false},
        {"text": "不正解の選択肢C", "isCorrect": false}
      ],
      "explanation": "解説：なぜその答えが正解なのか説明",
      "staffPoint": "患者様への説明ポイント（任意）"
    }
  ]
}

■ 問題作成ルール
1. 選択肢は4つ、正解は1つだけ
2. 難易度バランス: 基礎4問、応用4問、実践2問
3. 解説は100〜200文字程度
4. staffPointは全問に必要ではない（3〜4問程度でOK）
5. 紛らわしい誤答を含める（理解していないと間違えるもの）

■ 問題の種類（バランスよく含める）
- 定義・概念（「〇〇とは何ですか？」）
- 原因・メカニズム
- 症状・分類
- 診断・検査
- 治療・対処
- 患者対応・説明

参考ページの内容を読み取って、上記形式でクイズを作成してください。
```

## テーマ指定プロンプト（URLなし）

```
以下の仕様に従って、眼科スタッフ向けの「ドライアイ」クイズを作成してください。

■ クイズ情報
- タイトル: ドライアイクイズ
- 問題数: 10問
- 対象: 眼科クリニックのスタッフ（医師ではなく、受付・検査スタッフ向け）

■ 出力形式
以下のJSON形式で出力してください：

{
  "quizTitle": "ドライアイクイズ",
  "quizSubtitle": "眼科スタッフ向け - ドライアイの理解を深めよう",
  "quizIcon": "👁️",
  "quizId": "dryeye",
  "totalQuestions": 10,
  "questions": [
    {
      "id": 1,
      "questionText": "問題文",
      "options": [
        {"text": "正解の選択肢", "isCorrect": true},
        {"text": "不正解の選択肢A", "isCorrect": false},
        {"text": "不正解の選択肢B", "isCorrect": false},
        {"text": "不正解の選択肢C", "isCorrect": false}
      ],
      "explanation": "解説：なぜその答えが正解なのか説明",
      "staffPoint": "患者様への説明ポイント（任意）"
    }
  ]
}

■ 問題作成ルール
1. 選択肢は4つ、正解は1つだけ
2. 難易度: 専門医レベルではなく、スタッフが知っておくべき基礎〜応用
3. 解説は100〜200文字程度
4. staffPointは患者様への説明で使えるフレーズ（3〜4問程度）
5. 実際の患者対応で役立つ知識を重視

■ 含めてほしい内容
- ドライアイの定義と症状
- 原因（環境要因、加齢、VDT作業など）
- 診断方法（シルマーテスト、BUT等）
- 治療法（点眼薬、生活指導など）
- 患者様への説明ポイント

上記形式でクイズを作成してください。
```

## 簡易プロンプト（最短版）

```
眼科スタッフ向けの「[テーマ]」クイズを10問、以下のJSON形式で作成してください。
4択問題で、正解は1つ。各問題に解説をつけてください。

{
  "quizTitle": "タイトル",
  "quizIcon": "絵文字",
  "quizId": "英語ID",
  "questions": [
    {
      "id": 1,
      "questionText": "問題文",
      "options": [
        {"text": "選択肢", "isCorrect": true/false}
      ],
      "explanation": "解説文"
    }
  ]
}
```

## 既存クイズへの問題追加プロンプト

```
以下の参考ページから、既存クイズに追加する問題を5問作成してください。

■ 参考ページ
URL: [URL]

■ 既存クイズ情報
- クイズID: [既存のID]
- 現在の問題数: [現在の数]
- 追加後の問題数: [追加後の数]

■ 出力形式
questionsの配列のみ出力してください（id は [開始番号] から開始）：

"questions": [
  {
    "id": [開始番号],
    "questionText": "...",
    ...
  }
]

既存の問題と重複しない内容で作成してください。
```

## Tips

1. **URLがアクセスできない場合**
   - ページの内容をコピペして渡す
   - またはテーマ指定プロンプトを使用

2. **問題の質が低い場合**
   - 「もう少し難易度を上げてください」
   - 「患者対応に関する問題を増やしてください」
   - 「選択肢をもう少し紛らわしくしてください」

3. **出力が途中で切れた場合**
   - 「続きを出力してください」
   - 「問題6から10までを出力してください」

4. **修正を依頼する場合**
   - 「問題3の解説をもう少し詳しくしてください」
   - 「問題5の正解が間違っています。○○が正解です」
