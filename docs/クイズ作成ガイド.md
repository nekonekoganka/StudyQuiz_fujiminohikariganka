# クイズ作成ガイド（別AIへの指示用）

## 概要
眼科スタッフ向けの学習クイズを作成するためのガイドです。
参考ウェブページの内容を読み取り、以下の形式でクイズを作成してください。

---

## 出力形式

以下のJSON形式で問題を10問作成してください：

```json
{
  "quizTitle": "流涙症クイズ",
  "quizSubtitle": "眼科スタッフ向け - 流涙症の理解を深めよう",
  "quizIcon": "💧",
  "quizId": "ruiruisho",
  "questions": [
    {
      "id": 1,
      "questionText": "問題文をここに記載",
      "options": [
        {"text": "選択肢1（正解）", "isCorrect": true},
        {"text": "選択肢2", "isCorrect": false},
        {"text": "選択肢3", "isCorrect": false},
        {"text": "選択肢4", "isCorrect": false}
      ],
      "explanation": "解説文をここに記載。なぜその答えが正解なのか、背景知識も含めて説明。",
      "staffPoint": "患者様への説明ポイント（任意）：実際の患者様対応で使える説明の仕方やコツ"
    }
  ]
}
```

---

## 問題作成のルール

### 1. 問題数
- **10問**を作成

### 2. 選択肢
- 各問題に**4つの選択肢**
- **正解は1つだけ**（`isCorrect: true`）
- 選択肢は短く簡潔に（1〜2行程度）
- 紛らわしい誤答を含める（単純な間違いではなく、理解していないと間違えるもの）

### 3. 問題の難易度バランス
- 基礎問題：4問程度（定義、基本的な症状など）
- 応用問題：4問程度（原因、治療法、鑑別など）
- 実践問題：2問程度（患者様への説明、対応方法など）

### 4. 解説文
- 100〜200文字程度
- 「解説：」で始める
- なぜその答えが正解なのか明確に
- 関連する追加知識も記載

### 5. スタッフポイント（任意）
- 患者様への説明で使えるフレーズ
- 保護者への伝え方のコツ
- 実務で役立つアドバイス
- 全問に必要ではない（3〜4問程度でOK）

---

## 問題の種類（バリエーション）

以下の種類をバランスよく含めてください：

1. **定義・概念**
   - 「〇〇とは何ですか？」
   - 「〇〇の特徴として正しいのはどれですか？」

2. **原因・メカニズム**
   - 「〇〇が起こる原因は？」
   - 「〇〇のメカニズムとして正しいのは？」

3. **症状・分類**
   - 「〇〇の主な症状は？」
   - 「〇〇の種類として正しくないのは？」

4. **診断・検査**
   - 「〇〇の診断に用いる検査は？」
   - 「〇〇を疑う所見として正しいのは？」

5. **治療・対処**
   - 「〇〇の治療法として適切なのは？」
   - 「〇〇に対する初期対応は？」

6. **患者対応・説明**
   - 「患者様に〇〇を説明する際、適切な表現は？」
   - 「〇〇を心配する患者様への対応として正しいのは？」

## 良い問題の例

```json
{
  "id": 1,
  "questionText": "流涙症の原因として「分泌性流涙」と「導涙性流涙」がありますが、涙道閉塞が原因となるのはどちらですか？",
  "options": [
    {"text": "導涙性流涙", "isCorrect": true},
    {"text": "分泌性流涙", "isCorrect": false},
    {"text": "どちらにも当てはまる", "isCorrect": false},
    {"text": "どちらにも当てはまらない", "isCorrect": false}
  ],
  "explanation": "解説：導涙性流涙は、涙の排出経路（涙道）に問題がある場合に起こります。涙道閉塞や狭窄により、正常に分泌された涙が排出されずに溢れてしまう状態です。一方、分泌性流涙は涙の過剰分泌が原因です。",
  "staffPoint": "患者様への説明ポイント：「涙の通り道が詰まって、排水がうまくいかない状態です。水道の排水口が詰まっているようなイメージです」と説明すると理解してもらいやすいです。"
}
```

---

## 避けるべき問題

- ❌ 答えが明らかすぎる問題
- ❌ 「〇〇でないものはどれか」形式ばかり
- ❌ 数字の暗記を問う問題（臨床的に重要でない限り）
- ❌ 最新のガイドラインでは異なる古い情報
- ❌ 専門医レベルの難問（スタッフ向けなので）

---

## 参考ページの読み取り方

1. **見出し・小見出し**を確認し、主要トピックを把握
2. **定義・概要**部分から基礎問題を作成
3. **原因・症状**部分から応用問題を作成
4. **治療・対処法**部分から実践問題を作成
5. **患者様への説明**に使えそうな表現をスタッフポイントに活用
